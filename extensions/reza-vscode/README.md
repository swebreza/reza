# Reza — Code Graph

Interactive code knowledge graph with live session awareness. Requires [reza](https://github.com/swebreza/reza) CLI and a project with `.reza/context.db`.

## Commands

- **Reza: Show Code Graph** — Open the graph panel
- **Reza: Show Blast Radius** — Impact for the active file
- **Reza: Build / Refresh Graph** — Run `reza graph build`

See the main [reza documentation](https://github.com/swebreza/reza) for setup.
